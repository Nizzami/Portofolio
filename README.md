## ⚙️ Cara Menjalankan Project

1. Clone repository

```bash
git clone https://github.com/USERNAME/portfolio.git
```

2. Masuk ke folder project

```bash
cd portfolio
```

3. Install dependencies

```bash
composer install
```

4. Jalankan server Laravel

```bash
php artisan serve
```

5. Buka di browser

```
http://127.0.0.1:8000
```
